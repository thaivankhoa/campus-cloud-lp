# Campus Cloud - Landing Page

Landing page cho giải pháp quản lý giáo dục Campus Cloud.

## Cấu trúc thư mục

```
.
├── index.html          # Trang chủ
├── css/
│   └── styles.css     # Stylesheet chính
├── js/
│   └── script.js      # JavaScript
└── images/
    ├── logo.png       # Logo
    └── screenshot*.jpg # Screenshots
```

## Deploy lên GitHub Pages

1. Push code lên GitHub repository
2. Vào Settings > Pages
3. Chọn branch `main` (hoặc `master`)
4. Chọn folder `/ (root)`
5. Save và đợi vài phút để GitHub Pages build

Website sẽ có địa chỉ: `https://[username].github.io/[repository-name]`
